# **App Name**: AuthNexus

## Core Features:

- Login: Authenticate users using email and password. Firebase manages authentication.
- Sign-up: Allow new users to create accounts with email and password.
- Forgot Password: Enable users to reset their password via email.
- Loading Screen: Show a full-screen loading animation while the app initializes or fetches data.
- Error Messaging: Display user-friendly error messages for login failures, such as invalid credentials or network issues.

## Style Guidelines:

- Primary color: Cyan (#00BCD4) for a modern and trustworthy feel.
- Background color: Dark gray (#262A2B) to ensure sufficient contrast for the light text elements in dark mode.
- Accent color: Blue (#3F51B5) to highlight interactive elements.
- Body and headline font: 'Inter' sans-serif font to give a modern, machined, objective, neutral look.
- Use simple, line-based icons from Lucide React for visual clarity.
- Center the login form on the page with a max-width container for readability.
- Use subtle animations like fade-ins and loading spinners to enhance the user experience.
